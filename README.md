# python-web-tornado-hello-world

## Description
Simple web app for a tornado project.

## Tech stack
- python

## Docker stack

## To run
`sudo ./install.sh -u`
- http://localhost

## To stop (optional)
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
